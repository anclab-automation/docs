## Get Up and Running

Follow these steps to go from zero to querying your first knowledge graph.

<Steps>
  <Step title="Create Your Account">
    Sign up at [anclab.io](https://anclab.io) and generate an API key from the **Settings → API Keys** page. Your key will have the prefix `anchub_sk_`.
  </Step>
  <Step title="Model Your Ontology">
    Define the classes, properties, and constraints that describe your domain using the Ontology API or the visual ontology editor in the dashboard.
  </Step>
  <Step title="Build Your Knowledge Graph">
    Add entities and relationships that instantiate your ontology. Import existing RDF datasets or create triples directly through the API.
  </Step>
  <Step title="Query and Analyze">
    Run SPARQL queries against your graph, apply inference rules, and open the Analytics dashboard to visualize patterns and derive insights.
  </Step>
</Steps>

<Note>
  New to knowledge graphs? Start with the [Introduction](/introduction) to understand the core concepts, then follow the [Quickstart](/quickstart) to build your first graph.
</Note>
